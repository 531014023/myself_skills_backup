# 巴菲特思维框架 · Skill文档

## 概述

本skill旨在帮助用户以巴菲特的视角分析投资决策和商业判断。

## 核心功能

### 1. 投资分析
- 用巴菲特框架评估投资标的
- 分析护城河、估值、买入时机
- 提供决策建议

### 2. 案例解读
- 解析经典投资案例
- 分析失败案例的教训
- 提供可借鉴的思维模式

### 3. 商业判断
- 评估管理层质量
- 分析护城河来源
- 判断竞争优势持久性

## 触发方式

当用户提到以下关键词时触发：
- 「用巴菲特的视角」
- 「巴菲特会怎么看」
- 「Buffett模式」
- 「buffett perspective」
- 「帮我用巴菲特的角度想想」
- 「如果巴菲特会怎么做」

## 使用建议

1. **明确问题**：尽量具体地描述你的投资困惑
2. **提供背景**：如果有具体公司或行业，提供相关信息
3. **多轮对话**：不要期待一次性得到完整答案，可以追问

## 注意事项

- 本skill提供的是巴菲特的思维方式，不是投资建议
- 投资决策需结合个人情况，谨慎判断
- 巴菲特的框架更适合长期投资，而非短期交易
