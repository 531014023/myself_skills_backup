# 巴菲特 · 持仓与决策

## 一、持仓逻辑

### 核心持仓原则
1. **质地优先**：好公司 > 好价格 > 好时机
2. **长期持有**：近乎永久持有优质公司
3. **集中持仓**：高确信度下集中投资
4. **能力圈**：只投资自己能理解的公司

### 决策流程
```
① 发现机会 → ② 评估护城河 → ③ 计算内在价值 → ④ 等待安全边际 → ⑤ 买入 → ⑥ 持有
```

## 二、重大投资案例

### 成功案例

| 公司 | 买入时间 | 投资金额 | 核心逻辑 | 结果 |
|------|---------|---------|---------|------|
| 可口可乐 | 1988 | 13亿 | 全球最强品牌，护城河宽广 | 持有至今，回报超20倍 |
| 美国运通 | 1964/1991 | ~1300万/3亿 | 信用卡网络效应 | 持有至今 |
| 华盛顿邮报 | 1973 | ~1000万 | 垄断+低估 | 约20倍回报 |
| 富国银行 | 1990 | 2.9亿 | 稳健银行，管理层优秀 | 持有25年 |
| 苹果 | 2016-18 | 360亿 | 生态系统护城河 | 成为最大持仓 |
| 比亚迪 | 2008 | 2.3亿 | 新能源龙头（李录推荐） | 12年赚30倍 |

### 失败/教训案例

| 公司 | 买入时间 | 损失 | 教训 |
|------|---------|------|------|
| 德克斯特鞋业 | 1993 | 全部损失 | 跨国并购风险，被中国竞争摧毁 |
| 康菲石油 | 2008 | 数十亿 | 不该追逐大宗商品周期 |
| IBM | 2011-17 | 退出时亏损 | 科技护城河难维持 |
| 全美航空 | 1989 | 几乎归零 | 烂行业+烂管理层 |

## 三、买卖原则

### 买入原则
1. **好公司+合理价格**：从"捡烟蒂"进化到"买优质"
2. **安全边际**：好公司也要有好价格
3. **机会成本**：资金有限，优中选优
4. **长期视角**：做好持有10年以上的准备

### 卖出原则（很少发生）
1. 公司基本面恶化
2. 发现更好的机会
3. 估值极端泡沫
4. **不加仓亏损股**：从不摊平成本

### 不卖的理由
- 优秀的公司值得永远持有
- 交易产生摩擦成本
- 市场短期波动不重要

## 四、投资禁忌

### 巴菲特的"七不投"
1. **不投不懂的公司**
2. **不投没有护城河的公司**
3. **不投管理层不诚信的公司**
4. **不投周期性顶峰的强周期股**
5. **不投高杠杆公司**
6. **不投科技股（护城河建立后除外，如苹果）**
7. **不加码亏损投资**

## 五、决策框架

### 内在价值评估
```
内在价值 = 未来自由现金流的折现
```

**简化版（适用于稳定现金流公司）**：
```
估值 = EPS × (8.5 + 2G)
其中G = 未来7-10年的预期增长率
```

### 护城河评估清单
- [ ] 无形资产（品牌、专利、特许经营权）
- [ ] 转换成本（客户换竞品的代价）
- [ ] 网络效应（用户越多越有价值）
- [ ] 成本优势（流程、地点、资源）
- [ ] 规模效应（单位成本随规模下降）

## 六、重大决策回顾

### 1990s：从"捡烟蒂"到"买优质"
- 芒格说服巴菲特：好公司在合理价格买入，优于普通公司在便宜价格买入
- 代表案例：可口可乐、华盛顿邮报

### 2000s：金融危机中的逆向投资
- 2008年高盛优先股：10%股息+认股权证
- "别人恐惧时贪婪"

### 2010s：拥抱科技
- 2016年开始买入苹果
- 理由：消费者产品+科技属性+强大生态系统

### 2020s：大象级收购
- 2020年：西方石油（能源）
- 2022年：Alleghany保险
- 现金储备创历史新高，等待大机会
