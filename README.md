# myself_skills_backup

自己创建的一些 Skills 备份，防止丢失。所有 Skill 旨在供个人使用的 AI 辅助工具。

## Skill 列表

| Skill | 用途 |
|---|---|
| [annual-report-extractor](./annual-report-extractor/) | 从年报 PDF 提取财务数据和产能数据，支持自然语言指令自动解析公司名称、年份范围和数据字段 |
| [buffett-perspective](./buffett-perspective/) | 沃伦·巴菲特的思维框架与表达方式，基于致股东的信（1965年至今）和公开演讲 |
| [cninfo-annual-download](./cninfo-annual-download/) | 巨潮资讯网年报下载器，支持 A股（沪/深/北交所）和港股的年报、半年报、季报等定期报告下载 |
| [hotlist-fetcher](./hotlist-fetcher/) | 抓取今日热榜（tophub.today）数据并保存为 Markdown，可对接 Obsidian 归档 |
| [huashu-nuwa](./huashu-nuwa/) | 女娲造人：输入人名/主题/模糊需求，自动深度调研→思维框架提炼→生成可运行的人物 Skill |
| [munger-perspective](./munger-perspective/) | 查理·芒格的思维框架与表达方式，基于《穷查理宝典》、股东会讲话、哈佛/USC 演讲等 120+ 来源 |
| [personal-rules](./personal-rules/) | 全域双轨专家级回答框架：权威信源、实时时效验证、跨学科视角、多声部争议、三重交叉验证、溯源坐标 |
| [personal-writing-style](./personal-writing-style/) | 投资分析个人写作风格技能：第一人称叙事、结构化逻辑、数据驱动与定性结合、直接坦率 |
| [tang-chao-perspective](./tang-chao-perspective/) | 唐朝（老唐）的价值投资思维框架，基于老唐文集六册（246万字）提炼5个核心心智模型和10条决策启发式 |
| [value-investment-debate](./value-investment-debate/) | 价值投资多角度辩论分析工具，对投资标的、商业决策进行深度多角度分析 |
| [wechat-article-fetcher](./wechat-article-fetcher/) | 微信公众号文章抓取工具，支持批量抓取、图片下载、Markdown/HTML 格式转换和归档备份 |
