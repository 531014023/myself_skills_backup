# Hotlist Fetcher Skill

## 用途

自动抓取 https://tophub.today/hot 的热榜数据，整理成结构化的 Markdown 文档。

## 触发条件

当用户提到以下关键词时触发此 skill：
- "抓取热榜"
- "获取今日热榜"
- "tophub 热榜"
- "热榜数据"
- "保存热榜"
- "每日热榜"
- "今日热点"

## 工作流程

### 步骤 1: 抓取热榜数据

使用 MCP fetcher-remote 工具抓取热榜页面：

```
工具: fetcher-remote/fetch_url
参数: {
  "url": "https://tophub.today/hot",
  "extractContent": true,
  "maxLength": 15000
}
```

### 步骤 2: 解析数据

从返回的内容中提取以下信息：
- 排名
- 平台（知乎、微博、虎扑等）
- 标题
- 热度数值
- 链接

### 步骤 3: 生成 Markdown 文档

创建包含以下内容的 Markdown 文件：
- 标题和日期
- 数据来源说明
- TOP 30 热榜表格
- 平台分布统计
- 热点分类（商业财经、科技数码、国际局势、社会民生等）
- 选题建议

### 步骤 4: 保存文件

保存到用户指定目录，默认路径：`热榜/YYYYMMDD_今日热榜.md`

文件名格式：`{日期}_今日热榜.md`

## 注意事项

1. 抓取时设置 `maxLength` 为 15000 以获取完整内容
2. 提取 TOP 30 热点即可，但要保证至少有30条数据
3. 平台名称统一为：知乎、微博、虎扑、微信、虎嗅网、IT之家等
4. 热点分类要准确，便于用户选题参考
5. 文件编码使用 UTF-8
