## 一、1995年离开小霸王，创办步步高

### 背景

段永平1989年接手中山怡华集团下属一家亏损200万的小厂，至1995年，小霸王产值已逾10亿元，市场份额逼近80%，是最炙手可热的消费电子品牌。邀请成龙代言、"望子成龙小霸王"广告语家喻户晓。

### 决策：为何离开

**他本人陈述的理由**（核心关键词：契约）：

> "最早大老板说三七开，后来改二八了，后来说一九了。那招人的时候我跟弟兄们都是有承诺的，最后发现没办法兑现这个承诺……我觉得这个肯定不行，所以我就决定不干了。"
>
> "没有契约你就不可信了，再讲什么都是不可信的。你有一次两次可以，难道你还要第三次？就像刮瓶盖刮出'谢'了，你还要继续刮完吗？两个字刮出来一个字，你就肯定就不往下刮了。我就是刮到那个字了。"

**客观补充背景**：
- 段永平曾多次向上级公司提出股份制改造，让员工共享企业发展成果，据报道他曾提出拿出4亿元实施员工持股。
- 上级公司对股份制改造方案持警惕和反对态度，最终未获通过。
- 离开时，怡华集团老板陈健仁为他办了欢送酒会，送了一辆奔驰，条件是"一年内不与小霸王竞争"。

### 逻辑分析

| 维度 | 分析 |
|------|------|
| **价值观驱动** | "本分"价值观的第一次公开实践——承诺无法兑现，就不能继续 |
| **机会成本判断** | 小霸王已是顶锋，但机制的天花板已经看得到 |
| **离开成本低** | 核心团队6人愿意追随，研发和渠道可以带走重建 |
| **创业时机** | 1995年国内消费电子市场仍在高速增长期，机会窗口敞开 |

### 结果

1995年9月18日，东莞步步高电子有限公司成立。5年内，步步高成为无绳电话、VCD领域的头部品牌，并在之后孵化出OPPO、vivo、小天才三个独立品牌。

### 决策本质

**"做对的事情" vs "把事情做对"** ——这是段永平此后40年最重要的决策框架首次亮相：发现方向错了（机制问题），不是修补，而是果断停手、重新开始。

## 二、1999年拆分步步高：OPPO/vivo/小天才分家

### 背景

1995-1999年，步步高已形成三大业务板块：
- **教育电子**：点读机、学习机等（金志江负责）
- **视听电子**：VCD、DVD、MP3等（陈明永负责）
- **通讯科技**：无绳电话、手机等（沈炜负责）

三大业务同时运营，段永平本人精力有限，且各业务面临的竞争环境和技术路径差异越来越大。

### 决策：为何分家

**他本人陈述的理由**（核心关键词：授权与激励）：

- 借鉴硅谷模式，将股权分给真正在一线打仗的人
- 将自己持有的约70%股份大幅稀释至约10%，其余分给各业务负责人
- 原则：**人随事走，股权独立，互无从属**

**客观分析**：

1. **业务差异性**：教育、视听、通讯三块业务的技术路径、渠道模式、竞争格局完全不同，继续绑在一起会导致决策效率低下
2. **人才激励**：三个负责人（陈明永、沈炜、金志江）的能力已足以独当一面，需要更大的自主权和利益绑定
3. **风险隔离**：万一某一业务出现重大危机，不至于波及其他业务

### 关键机制设计

```
步步高（段永平控制，约10%股份）
    ├── 步步高教育电子 → 金志江 → 小天才
    ├── 步步高视听电子 → 陈明永 → OPPO（2001年陈明永购买OPPO品牌权）
    └── 步步高通讯科技 → 沈炜 → vivo（2011年推出vivo品牌）
```

段永平设计了一个**动态股权池**：三家独立公司之间有少量交叉持股（约10%），但各自独立运营、独立上市路径。

### 结果

- 陈明永在2004年成立OPPO（中国），2011年OPPO进入智能手机赛道
- 沈炜在2011年推出vivo品牌，专注于音乐手机定位
- 金志江将步步高教育电子发展成小天才（电话手表等）
- OPPO和vivo在2016-2018年合计占据中国智能手机市场约40%份额

### 决策本质

这是段永平**"本分"价值观的企业制度化**——真正的授权不是口头上的，而是把股权真的分出去。"放手"不是放弃，而是用制度代替个人控制。

## 三、2001年移居美国：急流勇退

### 背景

2001年，段永平40岁，步步高三家公司已拆分完毕，正在各自成长。此时，他拿到了美国绿卡（其妻刘昕在美国有事业基础）。

### 决策：为何急流勇退

**他本人陈述的理由**（核心关键词：家庭与承诺）：

> "如果为了事业不要家庭，那这个家还有什么意义？"

**客观分析（多重因素叠加）**：

| 因素 | 说明 |
|------|------|
| **爱情/家庭因素** | 妻子刘昕（人民大学同学）一直在美国发展，段永平在38岁才结婚，承诺过会到美国团聚 |
| **实业转型投资** | 2001年互联网泡沫破裂，美股出现大量被错杀的公司，正是入场时机 |
| **制度已建立** | 三家公司拆分完毕，股权已分，团队已成熟，不再需要他日常管理 |
| **个人志趣** | 段永平本人多次表示"不喜欢高压生活"，喜欢高尔夫球等悠闲活动 |

**关键转折点**：他赴美的时机恰好是互联网泡沫破裂、网易股价跌至0.8美元的时刻。

### 临别赠言

段永平离开时对同事说：**"如果做得不好，就把企业关了，我不会再回来管。"**

这句话的意义：不是"我信任你们所以放手"，而是"我已经做好了最坏的制度安排，剩下的交给你们"。体现了段永平一贯的**风险底线思维**。

### 结果

- 步步高三家公司在此后20年持续壮大
- 段永平2001年以约1美元买入网易，最终持有8-9年，卖出价超过100美元，收益超100倍
- 段永平本人表示："过去五年在美国炒股赚到的钱，比此前在国内做十多年企业赚的钱还要多得多"

### 决策本质

**人生优先级排序**：家庭 > 事业 > 财富积累。40岁退休不是放弃，而是把人生的杠杆从"自己经营"切换到"资本+人才"。

## 四、投资网易：从"垃圾股"到百倍回报

### 背景

2000-2001年，网易面临**三重打击**：
1. 互联网泡沫破裂，纳斯达克指数2001年跌幅达78%
2. 美国证券交易委员会（SEC）质疑网易财报造假，面临集体诉讼
3. 主营业务（门户广告）持续亏损，1999年亏损5千万，2000年亏损2.3亿

网易股价从上市高点约15.5美元，跌至0.48美元，面临退市风险。

### 决策：为何买入网易

**他本人陈述的理由**（核心关键词：懂 + 价值）：

> "我从来没认为我投网易是逆向投资。我投资的标准非常简单，就是在我自己能理解的范围内，找长期回报较高的公司去持有。"

**尽调逻辑（段永平事后复盘）**：
- 账上现金每股约2美元，股价却不到1美元——严重低估
- 聘请律师评估诉讼风险，判断胜算较高
- 网易的游戏业务（2001年《大话西游》上线）显示出转型潜力
- 丁磊本人是技术出身，重视产品而非讲故事

### 买入操作

| 指标 | 数据 |
|------|------|
| 买入时间 | 2001年底 - 2002年4月 |
| 买入均价 | 约1美元 |
| 买入数量 | 152万股（后增持至205万股） |
| 占总股本 | 约6.8%（第一大个人股东） |
| 总投入 | 约200万美元（含借款100万） |

### 结果与持有策略

- 2003年10月，网易股价涨至70美元，一年多涨50倍
- 段永平持有网易约8-9年，期间多次面对波动未卖
- 最终在股价超过100美元后陆续卖出
- **总收益：超过100倍（约2.2-2.6亿美元）**

### 最重要的决策细节

段永平后来说：**"1美元买网易的人挺多的，但能持有到100美元的人只有我一个。"**

这句话点出了投资的真正难点：**买入和持有是两件完全不同的事**。前者需要研究，后者需要信念（来自于真正的理解）。

### 决策本质

**能力圈原则的首次大规模实践**：段永平在做步步高时便理解游戏/互联网产品，他"懂"网易的商业逻辑，这给了他持有的信心。

## 五、投资茅台/苹果/腾讯：投资理念体系形成

### 贵州茅台

**买入逻辑**：
- 高毛利率（约91%）、高净资产收益率（ROE常年30%+）
- 自主定价权——可以不断提价而不失客户
- 越陈越香，年份溢价，库存价值随时间增长
- 品牌护城河极深，白酒文化根植高端消费群体心智

**他的表述**：
> "在A股我只买了茅台。投资茅台不需要太看宏观环境，肯定比放银行好。"
> "决定投资不是由市盈率来决定的，是由你未来的现金流来决定。"

### 苹果（Apple）

**买入逻辑**：
- 强大的生态锁定效应（iOS用户转换成本极高）
- 强大的企业文化——乔布斯建立的产品文化
- 管理层的"定价权"思维

**他的表述**：
> "当苹果大跌时，一定会买。"

**持仓**：据估算，苹果占其美股组合约80%（2023年底美股总持仓约145亿美元）。

### 腾讯

**买入逻辑**：
- 高净利率、高ROE、高成长性的"三高"特征
- 微信的不可替代性——"用微信的人根本无法离开"
- 强大的产品经理文化
- 通过投资延伸"互联网+各行业"的生态布局

**他的表述**：
> "当腾讯大跌时，有时会买。"
> "我觉得我对苹果、腾讯、茅台这三家公司是比较懂的。"

### 投资理念核心提炼

| 原则 | 段永平的表达 |
|------|-------------|
| 能力圈 | "不懂不投" |
| 买公司思维 | "买股票就是买公司" |
| 估值思维 | "买未来现金流折现，不靠市盈率决定" |
| 集中持仓 | "我就三只股票：苹果、腾讯、茅台" |
| 不加杠杆 | "不要用杠杆，不要用股票抵押" |
| 下行空间思维 | "通过大致判断确定下行空间，而不是精确计算" |

### 决策本质

段永平的投资方法论是**"做企业"思维的延伸**：他在步步高做了20年企业，理解什么是好产品、好管理、好文化。投资不过是把这种判断力用到了二级市场。

## 六、创办/投资极兔/极米/拼多多：个人投资 vs 实业参与

### 拼多多

**角色**：核心天使投资人

- 2015年黄峥创立拼好货/拼多多，段永平是核心天使投资人
- 参与方式：带黄峥参加2006年巴菲特午餐（黄峥称"这顿饭改变了我一生"）
- 段永平提出的投资条件：**"你想让我投多少，你就再投多少"**（以身作则的信任机制）
- 黄峥后来表示段永平是"导师"，但两人对外以"朋友"相称

**他本人的表述**：
> "黄峥想把我拉进去做投资人。我就问他能挣钱吗？他说不知道，但用户成长非常快，很多人喜欢用，供应商也很开心。"

**态度**：支持，但不主导；相信人，不干预具体经营。

### 极兔速递（J&T Express）

**角色**：天使投资人

- 极兔由OPPO印尼市场负责人李杰创办，段永平参与了早期投资
- **他本人的表态**：
  > "确实投了，算是友情支持一下。同时表示：不喜欢物流的商业模式，不建议大家投这类公司。"
  > "一般不投资任何非上市公司，除非是自己认识很久的（至少10年以上），并且非常了解和信任的人做的公司。"

**态度**：明确区分"友情支持朋友创业"和"基于商业模式的投资判断"。

### 极米科技

- 极米科技创始人钟波为段永平门徒，步步高体系出身
- 极米后来成为投影仪行业龙头（科创板上市）

### 本质区分：个人投资 vs 实业参与

| 类型 | 代表案例 | 段永平的角色 | 干预程度 |
|------|---------|------------|---------|
| 财务投资 | 茅台、苹果、腾讯 | 纯粹股东 | 极低 |
| 天使投资（熟悉的人） | 拼多多、极米 | 导师+投资人 | 低，仅价值观影响 |
| 友情支持（熟悉的人） | 极兔 | 朋友+天使 | 极低，不参与经营 |
| 战略性退出 | OPPO、vivo、小天才 | 创始人→股东 | 制度化管理，不干预 |

**底层逻辑**：段永平将"信任"作为投资非上市公司的核心门槛（10年以上认识），但对于纯财务投资，则完全基于"懂不懂商业模式"。

## 七、2010年后淡出日常管理："本分"放手的最高境界

### 背景

OPPO、vivo在智能手机时代崛起，成为中国手机市场头部品牌。段永平早在2001年便已退居幕后，此后20年OPPO、vivo的成长主要依靠陈明永、沈炜两套班子。

### 决策：他如何真正"放手"

**核心原则：本分文化作为制度替代个人控制**

段永平建立的OPPO和vivo内部，有一条不成文的文化基因：**"本分"**。这个词被写进了OPPO和vivo的企业文化手册，是所有决策的最终参照。

**他本人对"本分"的定义**（多次在雪球和访谈中提及）：
- 做对的事情
- 发现错了就尽快改
- 不赚不该赚的钱
- 不占别人便宜

**实际表现**：
- 段永平每年仅回国两次参加董事会
- OPPO/vivo重大战略决策（如2011年决定做智能手机、2018年Find X系列转型）均为陈明永/沈炜团队独立判断
- 段永平对具体业务决策"不了解，也不想了解"
- 2021年"OPPO造车"传闻，段永平在雪球回复：**"'重出江湖'的事情绝对不会发生。"**

### 标志性事件：段永平录制访谈中途离场

2025年，有主持人在访谈中问OPPO是否已背离"本分"，段永平一言不发，放下麦克风直接离场。

**事后的表态**："有些采访不如不采。"

这体现了段永平一贯的态度：**我只对我信任的人和事负责；我不为别人的判断背书。**

### 结果

OPPO、vivo在全球市场经历了：
- 2016-2018年：在中国市场合计份额约40%
- 2019年后：受到华为、小米等竞争，份额有所下降
- 2022年后：OPPO在欧洲遭遇专利战；vivo在印度市场遭遇监管挑战
- 整体：两家公司的企业文化和治理结构保持了相对稳定

### 决策本质

**"本分"价值观作为企业治理的替代机制**——当创始人无法（或不愿意）用个人权威管理企业时，最好的办法是建立一套内化的价值观体系，让团队在无人监督时也能做出正确决策。

## 八、综合：决策逻辑的一致性 vs 变化

### 贯穿始终的决策基因

| 维度 | 一致性表现 |
|------|-----------|
| **"做对的事情"框架** | 离开小霸王（机制错）、拆分步步高（授权对）、移居美国（优先级对）、不投不懂的公司 |
| **契约精神/本分** | 离开小霸王的直接原因；OPPO/vivo分股的本质；不干预徒弟的经营 |
| **风险底线思维** | 离开小霸王前谈好不竞争条款；移居美国时说"做不好就关掉"；不投物流商业模式 |
| **信任前置原则** | 对陈明永/沈炜的股权授予；黄峥的"友情投资"条件；极兔的"10年朋友"门槛 |
| **能力圈约束** | "不懂不投"；重仓苹果/茅台/腾讯而非分散 |

### 决策的变化维度

| 维度 | 变化表现 |
|------|---------|
| **杠杆使用** | 投资网易时曾借款100万（早期激进）→ 后期明确"不用杠杆" |
| **决策频率** | 从步步高时代每年做多个品类决策 → 投资时代每年甚至数年做一次重大决策 |
| **组织形态** | 从亲自下场经营 → 退居投资人 → 影响力输出（通过门徒） |
| **投资边界** | 从专注消费电子 → 互联网（网易） → 消费品（茅台） → 科技（苹果）→ 全类别 |

### 决策质量的评估框架

段永平40年决策史，可以用他自己的话做一个总结：

> **"做对的事情，把事情做对。"**

- **离开小霸王**：做对了"做对的事情"（发现机制问题，果断停手）
- **拆分步步高**：做对了"做对的事情"（授权给对的人）
- **投资网易**：做对了"把事情做对"（深入研究、坚定持有）
- **持有苹果茅台腾讯**：同时做对了两件事（对的标的 + 对的持有方式）
